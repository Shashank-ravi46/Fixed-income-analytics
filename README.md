# Fixed Income Analytics Suite

Python-based fixed income analytics platform built in Google Colab.

## What This Project Does
- Downloads real US Treasury yield data from the Federal Reserve (FRED API)
- Prices bonds and computes duration, convexity, and DV01
- Runs stress tests across nine yield shock scenarios
- Builds a five-bond portfolio with aggregate risk metrics
- Exports results to Excel and interactive Plotly charts

## Tools Used
Python | Google Colab | FRED API | Pandas | NumPy | Plotly | OpenPyXL

## Key Output
- 4-sheet Excel workbook
- 3 interactive HTML charts
- Portfolio DV01 and stress test P&L table
